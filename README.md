# oscomp-treeland-screencopy
### 项目名称

`xdg-desktop-portal` 支持 `treeland` 截图

### 项目描述

基于 `wlroots` 的 `Wayland Compositor` 通常支持应用程序屏幕截图。但是 `treeland` 暂时尚未完成。在此项目中，`treeland` 实现了 `XDG-DESKTOP-PORTAP-PORTAL` 屏幕截图协议以支持屏幕截图。
